# Mídias SAT TELECOM

| Arquivo | Uso |
|---------|-----|
| **`logo-quadrada.jpeg`** | Ícone do app (Android/iOS), splash e favicon — **não** aparece nas telas |
| `logo.png` | Logo horizontal para telas (login, home) — **adicionar quando disponível** |

## Ícone do app

O arquivo `logo-quadrada.jpeg` fica em:
- `assets/logo/logo-quadrada.jpeg`
- `apps/mobile-expo/assets/logo-quadrada.jpeg`

## Logo nas telas

Por enquanto o app exibe apenas o texto **SAT TELECOM** no header. Quando tiver a logo horizontal, coloque em `assets/logo/logo.png` e avise para integrar no app.
