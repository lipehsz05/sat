# Mídias — App Provedor SAT

Coloque aqui os arquivos de mídia usados pelo app React (Expo).

## Estrutura

| Pasta | Arquivo sugerido | Dimensões recomendadas |
|-------|------------------|------------------------|
| `logo/` | `logo.png` | 512×512 px (PNG transparente) |
| `banners/` | `home-banner.png` | 1200×400 px |
| `icons/` | ícones customizados | 96×96 px |

## Como usar

1. Substitua os placeholders pelos arquivos reais da sua marca.
2. Mantenha os mesmos nomes de arquivo para não precisar alterar o código.
3. O app referencia esta pasta via caminho relativo (`../../assets/`).
